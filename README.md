## Tiếng Việt

Repo này dùng để **lưu trữ lời giải LeetCode bằng C++**.

### Tự động đồng bộ (Auto-push)

Các bài giải trong repo này được **tự động push** từ repo/extension:

- `nkdkhtl/leetcode-to-github` — *This extension is vibecoded by Nam Khuc.*

> Nói ngắn gọn: bạn giải bài trên LeetCode → extension thực hiện commit/push → bài xuất hiện trong repo này.

### Cấu trúc thư mục

- `Solution/` — chứa các bài giải (mỗi bài là một file C++ hoặc một thư mục con tùy cách bạn tổ chức)

Ví dụ:
```text
Solution/
  0001-Two-Sum.cpp
  0020-Valid-Parentheses.cpp
  ...
```

### Mục tiêu

- Lưu lại quá trình luyện tập & ôn tập thuật toán
- Dễ tra cứu lại các dạng bài / kỹ thuật đã làm

## English

This repository is a **collection of LeetCode solutions written in C++**.

### Auto sync (Auto-push)

Solutions in this repository are **automatically pushed** via:

- `nkdkhtl/leetcode-to-github` — *This extension is vibecoded by Nam Khuc.*

> In short: solve on LeetCode → the extension commits/pushes → solutions appear here.

### Directory structure

- `Solution/` — contains the solutions (each problem is stored as a C++ file or subfolder depending on your organization)

Example:
```text
Solution/
  0001-Two-Sum.cpp
  0020-Valid-Parentheses.cpp
  ...
```

### Purpose

- Track progress and practice history
- Quickly revisit patterns and techniques
